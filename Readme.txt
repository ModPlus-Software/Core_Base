Если статус документа "Действующий", то должен быть атрибут DocStatus="True". Для недействующих соответственно DocStatus="False". 
А если статус "Не определен законодательством", то нужно удалить этот атрибут вовсе

При добавлении новых документов см.файл - "AddingToDb.xlsx"

Дополнительные параметры задаются атрибутом ItemTypeN (где N - номер по порядку). Атрибут должен содержать строку значений,
разделенных символом $, где первое значение - название параметра.
Например, ItemType1="Марка стали$Ст2кп$Ст2пс$Ст2сп$Ст3кп$Ст3пс$Ст3сп$Ст3пс3$Ст3сп3$Ст3пс4$Ст3сп4"
